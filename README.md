## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/spriti04/My-Portfolio.git
```

Move into the project directory:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run the project:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
